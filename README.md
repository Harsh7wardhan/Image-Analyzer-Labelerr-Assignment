# Labellerr-Image-Analyzer

Web App that allows users to upload an image and analyze it using the Computer Vision API.

## Configuration
1. Create a `.env` file and update it with your Computer Vision API subscription key and endpoint:
   SUBSCRIPTION_KEY=your_subscription_key
   ENDPOINT=your_endpoint

2. As the Google and Azure API's required card details , I have not used the actual API key and endpoints , but I have provided the entire implementation in the code , of how we would proceed given the key and endpoint .

## Running the Application

1. Start the development server: npm start
   
2. Open your web browser and go to http://localhost:3000
   
3. Click "Upload Image" button and you will get the analysis results.
